# Movieslike — vibe-based movie search

Live demo of [github.com/rahul30uic/movieslike](https://github.com/rahul30uic/movieslike):
movie discovery built from 4,400 Reddit "movies that feel like this?" posts,
served from a hybrid embedding space (VLM vibe captions + SigLIP image
vectors). The JSON API used by the frontend lives at `/recommendations`,
`/recommendations/text`, `/recommendations/image` (docs at `/docs`).
